# B站视频自动转文稿助手

这是一个基于 **阿里 SenseVoiceSmall** 的全自动B站视频转文字工具。

## 核心功能
* **自动监听触发**：只需复制链接（如 `https://www.bilibili.com/video/BV...`），无需手动粘贴。
* **无缝 AI 衔接**：自动合成结构化润色指令，一键跳转豆包 AI，粘贴即得精美文稿。

## 如何使用
### 第零步：安装Python

### 第一步：安装环境 (仅需一次)

管理员模式运行 **`一键安装环境.bat`**。

> 程序会自动配置 Python 依赖、CUDA 12.1 加速环境。
> **注意**：Python环境与第三方库下载在C盘，模型默认下载至 `D:/AI_Models_Cache`

### 第二步：启动助手

双击运行 **`启动程序.bat`**。

> 当看到 ` 长视频转文字工具 - 监听中` 时，助手已就绪。

### 第三步：复制链接

前往浏览器复制 B 站地址，剩下的交给助手：

1. **自动预处理**：音频会被自动重采样为 16kHz 单声道
2. **转录中**：命令行会显示转录进度（40分钟视频约 2-4 分钟转完）。
3. **唤醒 AI**：完成后自动打开豆包官网，你只需 **Ctrl + V** 即可开始 AI 总结。

##  注意事项

1. **显存保护**：处理超长视频时请关闭浏览器或 3D 游戏以预留空间。
2. **首次运行**：第一次转录会从 ModelScope（国内镜像）下载约 2GB 的模型文件，国内网络通常 2 分钟内完成。
3. **中文路径**：请确保本程序所在的文件夹路径不包含中文，以防 FFmpeg 读写失败。

#  技术路线

1. **音频提取与预处理**
* `yt-dlp`
* `FFmpeg`


2. **核心识别引擎**
* `SenseVoiceSmall`：多语言语音理解模型（330M 参数）。
* `fsmn-vad`：语音端点检测，防止长音频内存堆积。
* `ct-punc`：智能标点恢复模型。


3. **自动化链路**
* `pyperclip`：双向剪贴板交互。
* `ModelScope`：国内模型分发与缓存管理。
