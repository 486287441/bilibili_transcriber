#  B站视频自动转文稿助手

这是一个基于 OpenAI Whisper 的自动化工具。只需复制 B 站链接，程序会自动下载音频、识别文字、整理 Prompt，并为你打开豆包 AI 进行润色。

## 🌟 核心功能
* **自动监听**：只需复制链接（如 `https://www.bilibili.com/video/BV...`），无需手动粘贴。
* **AI 转写**：内置 OpenAI Whisper 模型，支持中英文精准识别。
* **硬件加速**：针对 NVIDIA 显卡（如 RTX 4060）深度优化，转写倍速高达 10x 以上。
* **无缝衔接**：自动合成润色指令，一键跳转豆包 AI 网页版。

## 📂 压缩包内容
* `bilibili transcriber.py` - 程序核心脚本
* `ffmpeg.exe` / `ffprobe.exe` - 音频处理核心（已内置，无需配置环境）
* `一键安装环境.bat` - **初次使用点我**
* `启动程序.bat` - **日常使用点我**

## 🛠️ 如何使用
### 第一步：安装环境 (仅需执行一次)
双击运行 **`一键安装环境.bat`**。
> 程序会自动安装 Python 依赖及 CUDA 显卡驱动或CPU驱动环境。请耐心等待进度条走完。

### 第二步：启动助手
双击运行 **`启动程序.bat`**。
> 当看到 `🎧 已就绪，请复制 B 站链接...` 时，说明程序已在后台开始监听。

### 第三步：复制链接
前往浏览器或 B 站客户端，**直接复制视频地址**。
1.  程序会自动开始下载并转录（你可以看到进度条）。
2.  完成后，浏览器会自动打开豆包官网。
3.  你只需在对话框 **Ctrl + V (粘贴)**，AI 就会开始为你整理文稿。

## ⚠️ 注意事项
1.  **首次运行**：第一次处理视频时，程序会静默下载识别模型（约 460MB），根据网络情况可能需要 3-5 分钟，之后使用将是秒开。
2.  **显卡要求**：本程序优先调用 NVIDIA 显卡。若无显卡，程序会自动切换至 CPU 模式，速度会稍慢。
3.  **保持窗口开启**：程序运行期间请勿关闭黑色的命令行窗口。
